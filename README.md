# Biblioteket i Fulköping

### Projektets syfte:

    Ett bibliotekssystem där användare kan låna och återlämna böcker, samt se sin lånehistorik och lånestatus.

### Förutsättningar:

    MySQL Database: phpMyAdmin 
    Importera SQL fil: "dblibrary.sql"
